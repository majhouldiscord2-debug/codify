# 🚀 CODIFY Studio - AI Design & Coding Assistant

**Built for Majhoul - The Future of AI-Powered Development**

CODIFY Studio is a professional, modern, and adaptive dashboard designed to empower designers and coders. It features a unique "Learning Engine" that adapts to your communication style and preferences.

## ✨ Key Features

- **🧠 Adaptive AI Engine**: The more you talk, the more it learns. It adapts its tone (from professional to "partner") and remembers your design preferences.
- **🛠️ AI Specialist Modules**:
  - **Design AI**: Master UI/UX with minimalist-aware suggestions.
  - **Code AI**: Optimized for Python and JavaScript logic.
  - **Game AI**: Experimental module for mechanics and logic.
- **📁 Project Management**: Create and inspect projects with persistent `localStorage` memory.
- **🔍 Resource Finder**: Instantly find free assets (Icons, Photos, UI Kits, Game Dev).
- **💻 Pro Code Editor**: Integrated Prism.js for syntax-highlighted AI outputs.
- **📊 AI Adaptation Stats**: Real-time tracking of how the AI is evolving to match your skills.

## 🚀 Getting Started

1.  **Clone the Repo**: Upload these files to your GitHub.
2.  **Deploy**: Connect your GitHub to **Cloudflare Pages**.
3.  **Configure**: Set the "Root directory" to `Dev-Dashboard` (if applicable) and deploy!
